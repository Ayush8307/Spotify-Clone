# Spotify Clone

A responsive Spotify Clone built using the MERN Stack that allows users to stream music, manage playlists, and enjoy a modern music player interface.

## Tech Stack

* HTML
* CSS

## Features

* It is just the front-end of the Spotify Music

## Run Locally

```bash
git clone https://github.com/your-username/spotify-clone.git
cd spotify-clone
npm install
npm start
```
